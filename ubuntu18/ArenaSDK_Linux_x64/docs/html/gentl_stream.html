GenTL Stream Features¶

Node

Type (Access)

Description

StreamID

String (RO)

Device unique ID for the data stream, e.g. a GUID.

StreamType

Enumeration (RO)

Custom transport layer. The entries for this enumeration node are ‘GigEVision’, ‘CameraLink’, ‘CameraLinkHS’, ‘CoaXPress’, ‘USB3Vision’, and ‘Custom’.

StreamAnnouncedBufferCount

Integer (RO)

Number of announced (known) buffers on this stream. This value is volatile. It may change if additional buffers are announced and/or buffers are revoked by the GenTL Consumer.

StreamBufferHandlingMode

Enumeration (RW)

The application always gets the buffer from the tail of the Output Buffer Queue and re-queues any remaining ones. The entries for this enumeration node are ‘OldestFirst’, ‘OldestFirstOverwrite’, and ‘NewestOnly’.

StreamAnnounceBufferMinimum

Integer (RO)

Minimal number of buffers to announce to enable selected buffer handling mode.

StreamDeliveredFrameCount

Integer (RO)

Number of delivered frames since last acquisition start.

StreamLostFrameCount

Integer (RO)

Number of lost frames due to queue underrun. This number is initialized with zero at the time the stream is opened and incremented every time the data could not be acquired because there was no buffer in the input buffer pool.

StreamInputBufferCount

Integer (RO)

Number of buffers in the input buffer pool.

StreamOutputBufferCount

Integer (RO)

Number of buffers in the output buffer queue.

StreamStartedFrameCount

Integer (RO)

Number of frames started in the acquisition engine. This number is incremented every time a new buffer is started to be filled (data written to) regardless if the buffer is later delivered to the user or discarded for any reason. This number is initialized with 0 at at the time of the stream is opened. It is not reset until the stream is closed.

PayloadSize

Integer (RO)

Size of the expected data in bytes. Note that this feature “overwrites” the PayloadSize of the remote device, see also sections “Data Payload Delivery” and “Allocate Memory” of the GenICam GenTL standard.

StreamChunkCountMaximum

Integer (RO)

Maximum number of chunks to be expected in a buffer (can be used to allocate the array for the DSGetBufferChunkData function).

StreamBufferAlignment

Integer (RO)

Alignment size in bytes of the buffer passed to DSAnnounceBuffer.

StreamMissedImageCount

Integer (RO)

Number of missed images, resetting on each stream start.

StreamMissedPacketCount

Integer (RO)

Number of missed packets, resetting on each stream start.

StreamCumulativeMissedImageCount

Integer (RO)

Number of missed images, not resetting on each stream start.

StreamCumulativeIncompleteImageCount

Integer (RO)

Number of incomplete images, not resetting on each stream start.

StreamMaxNumResendRequestsPerImage

Integer (RW)

Maximum number of resend requests per image.
