class ImageParams¶

A container to hold and calculate image parameters.

ImageParams¶

Save::ImageParams::ImageParams()¶

An empty constructor. Initializes width, height, and bits per pixel to 0.

ImageParams¶

Save::ImageParams::ImageParams(size_t width, size_t height, size_t bitsPerPixel, bool topToBottom = true)¶

A constructor.

Parameters

  • width –

    • Type: size_t

    • Image width

  • height –

    • Type: size_t

    • Image height

  • bitsPerPixel –

    • Type: size_t

    • Number of bits per pixel

  • topToBottom –

    • Type: bool

    • Default: true

    • If true, image data is read/written top to bottom

    • Otherwise, bottom to top

    • Only applicable when saving JPEG and BMP

GetSize¶

virtual size_t Save::ImageParams::GetSize() const¶

GetSize calculates the size of the image. The size is calculated by multiplying the width, height, and bytes per pixel (bits per pixel divided by 8) and adding an extra byte if a remainder of bits remains.

Returns

  • Type: size_t

  • Image size

GetStride¶

virtual size_t Save::ImageParams::GetStride() const¶

GetStride calculates the stride of an image. The stride is calculated by multiplying the width by the number of bytes per pixel (bits per pixel divided by 8).

Returns

  • Type: size_t

  • Image stride/pitch
