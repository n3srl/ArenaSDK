GenTL Device Features¶

Node

Type (Access)

Description

DeviceID

String (RO)

Interface-wide unique identifier of this device.

DeviceSerialNumber

String (RO)

Serial number of the remote device.

DeviceUserID

String (RO)

User-programmable device identifier of the remote device.

DeviceVendorName

String (RO)

Name of the remote device vendor.

DeviceModelName

String (RO)

Name of the remote device model.

DeviceVersion

String (RO)

Name of the version of the remote device model.

DeviceType

Enumeration (RO)

Custom transport layer. The entries for this enumeration node are ‘GigEVision’, ‘CameraLink’, ‘CameraLinkHS’, ‘CoaXPress’, ‘USB3Vision’, and ‘Custom’.

DeviceDisplayName

String (RO)

User readable name of the device. If this is not defined in the device this should be “VENDOR MODEL (ID)”.

DeviceAccessStatus

Enumeration (RW)

Non-available devices. The entries for this enumeration node are ‘Unknown’, ‘ReadWrite’, ‘ReadOnly’, and ‘NoAccess’.

DeviceEventDataFormat

Enumeration (RO)

Custom. The entries for this enumeration node are ‘None’, ‘GigEVision’, and ‘Custom’.

GevDeviceIPAddress

Integer (RO)

Current IP address of the GVCP interface of the remote device.

GevDeviceSubnetMask

Integer (RO)

Current subnet mask of the GVCP interface of the remote device.

GevDeviceMACAddress

Integer (RO)

48-bit MAC address of the GVCP interface of the remote device.

GevDeviceGateway

Integer (RO)

Current gateway IP address of the GVCP interface of the remote device.

CcpSwitchoverKey

Integer (RW)

User-specified Ccp Switchover Key.

DeviceEndianessMechanism

Enumeration (RW)

Handling the device endianess according to GenICam Schema 1.1 and later. The entries for this enumeration node are ‘Legacy’ and ‘Standard’.

StreamSelector

Integer (RW)

Selector for the different stream channels.The selector is 0-based in order to match the index of the C interface.

StreamID

String (RO)

Device unique ID for the stream, e.g. a GUID.

ActionCommandDeviceKey

Integer (RW)

User-specified Action Command Device Key.

ActionCommandGroupKey

Integer (RW)

User-specified Action Command Group Key.

ActionCommandGroupMask

Integer (RW)

User-specified Action Command Group Mask.

ActionCommandExecuteTime

Integer (RW)

Time when the device will execute the command.
