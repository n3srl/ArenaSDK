EIncModeClass¶

EIncModeClass¶

class EIncModeClass¶

EIncModeClass is a static class based on GenApi classes of the same purpose. It translates from an increment mode enum (GenApi::EIncMode) to a string and back. For example, the following enum value can be translated to and from the following string representation:

  • enum: GenApi::fixedIncrement

  • string: “fixedIncrement”

Warning

  • Similar classes are mostly found under the GenApi namespace

FromString¶

static bool Arena::EIncModeClass::FromString(const GenICam::gcstring &ValueStr, GenApi::EIncMode *pValue)¶

FromString translates a string representation of an increment mode enum to its value. The second parameter is an out parameter.

Parameters

  • ValueStr –

    • Type: const GenICam::gcstring&

    • String representation to translate

  • pValue –

    • Type: GenApi::EIncMode*

    • Out parameter

    • Pointer to translated enum value

Returns

  • Type: bool

  • True if successful

  • Otherwise, false

ToString¶

static void Arena::EIncModeClass::ToString(GenICam::gcstring &ValueStr, GenApi::EIncMode Value)¶

ToString translates from an increment mode enum value to its string representation. The first parameter is an out parameter.

Parameters

  • ValueStr –

    • Type: GenICam::gcstring&

    • Pass-by-reference out parameter

    • Translated string representation

    • ”intfIUnknown” on failure

  • Value –

    • Type: GenApi::EIncMode

    • Enum value to translate

Returns

  • none
