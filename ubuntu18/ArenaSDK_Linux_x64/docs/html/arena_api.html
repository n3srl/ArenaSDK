Arena Python API¶

Arena Python API is a python wrapper to interact with ArenaSDK.

This package is built on top of ArenaC API.

For Windows Arena Python API is shipped with ArenaView.

For Linux download here: https://thinklucid.com/downloads-hub/ - refer to readme file inside .zip

Arena Python API prerequisites

  • Python >= 3.6.8

  • Pip package: pywin32>=224

pip install pywin32

Offline machines only: - look at offline_installation_dependencies.zip.

Arena Python API installation instructions for Windows

Default location of .whl:

"C:\ProgramData\Lucid Vision Labs\ArenaView\ArenaPy"

To install:

pip install arena_api-<>.<>.<>-py3-none-any.whl

Examples prerequisites:

  • Make sure arena_api package is installed

  • Some examples require other pip packages which can be found in requirements_win.txt. To install these packages run:

pip install -r requirements_win.txt

To run examples locate their root folder:

"C:\ProgramData\Lucid Vision Labs\Examples\src\Python Source Code Examples"

Run an example with python command passing a filename:

python <py_example_name>.py
