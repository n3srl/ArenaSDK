Read¶

void ArenaNET.FeatureStream.Read(String fileName)

Streams all selected features from a file to a node map (INodeMap).

Parameters
  • fileName: Name of the file to stream from

Write¶

void ArenaNET.FeatureStream.Write(String fileName)

Stream all selected features from a node map (INodeMap) to a file.

Parameters
  • fileName: Name of the file to stream to

Select¶

void ArenaNET.FeatureStream.Select(String featureName)

Adds a single streamable feature to the list of selected features to stream. If called for the first time, also sets an internal ‘select-all’ flag to false.

Parameters
  • featureName: Name of the feature to select
