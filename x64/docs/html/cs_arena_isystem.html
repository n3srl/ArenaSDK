virtual void ArenaNET.ISystem.RemoveUnicastDiscoveryDevice(String remoteIpAddress)

ForceIp

virtual void ArenaNET.ISystem.ForceIp(UInt64 macAddress, UInt64 ipAddress, UInt64 subnetMask, UInt64 defaultGateway)

Forces the device that matches the macAddress to a temporary new ipAddress, subnetMask and defaultGateway.

Parameters
  • macAddress: 48 bit mac address identifying the device to force the ip to
  • ipAddress: 32bit integer representing the IP address that is meant to be forced to the device
  • subnetMask: 32bit integer representing the subnetmask that is meant to be forced to the device
  • defaultGateway: 32 bit integer representing the gateway that is meant to be forced to the device
